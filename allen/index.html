介紹一下你自己和所做的工作**。** 🔗

我叫 Allen ，中文名不詳（基本在工作中不太用，哈哈哈）。剛畢業的時候我一直希望能夠做一名優雅的平面設計師，於是在廣告公司呆了 4 年，並沒有什麼作品出來，因為一直在這做 AE 相關的事情 :-p。後來的事情只與音樂相關，Jing.fm 聯合創始人，章魚音樂創始人，這一開始就停不下來，已經快 6 年了。

章魚音樂有兩個線上產品， 章魚FM ， 呼呼催眠大師 。線下我們有章魚現場，在 2014 年 9 月－ 2015 年 1 月我們舉辦 10 場的「 走唱西北 」現場演出。

章魚的用戶一直對催眠的音樂非常感興趣，於是我們今年4月的時候發佈了「呼呼催眠大師」，其實更應該歸類於助眠應用。運用腦波音頻和自然環境聲音的混搭，從生理上幫助大家更好的進入睡眠，並提供給孩子的助眠功能。

兒童節的時候，章魚FM 上線了 Web 版本 2.0，不用下載也可以聽啦。

你都在使用哪些硬件？ 🔗

數碼之器

我是個  Apple 的腦殘粉，幾乎所有的數碼硬件都是 Apple 系列的產品。

工作時主力是 14 年的 MacBook Pro 15''，還有一個 13 寸的 MBP 方便攜帶。一直相信 只有 Pro 才能把軟件運行的更好。順帶著 HHKB Pro 2 鍵盤是當我偶爾寫代碼，或者碼字的時候的好幫手， Magic Trackpad 2 也是非常好的利器，可以讓你的手更舒服的來控制光標的移動（確實遠遠超過第一代 Trackpad）。

iPhone 6s Plus ，用它主要是大，iPhone 已經用了 9 年了，完全習慣於蘋果的生態系統，沒法跳出了，不管幹什麼都離不開 iPhone。 Nexus 5 是唯一一台放在身邊的做測試的 Android 手機，基本不太用。 Blackberry Classic Q20 一直是放在 Wishlist 的一款情懷手機，物理全鍵盤的誘惑非常大，隨時感覺都要入手。

在家是 Airport Extreme 和 Airport Express  組成的無線網絡。 Apple TV 和 Airplay 是非常實用的功能，用 iPad mini 作為視頻源，或者通過  Airport Express 把音樂投到臥室的小音箱上。追求音質的時候還會用 iPod Classic + PreSonus Eris E5 監聽音箱一對 + M-Audio Fire 610 聲卡來聽音樂。

運動中用得最多的是 Apple Watch （總共使用天數大約不到一個月，因為電池不夠猛，我的消息提醒太多，再加上之前測試 2.0 Beta 的時候卡住了，结果到現在也沒能正常使用。）

軟件呢？ 🔗

Work:

  • Chrome / Gmail / Google Drive / Google Calendar
  • Sketch / Photoshop / Pixate Studio
  • MindNode
  • Slack

一直在 Google Apps 裡面打轉， Gmail 來做郵件服務，並且現在不用任何的郵件客戶端，只是在  Chrome 上面登陸 Gmail Web 來使用。用  Google Drive 共享文件和協作文檔， Google Calendar 來 book 大家的時間，讓開會更便利。

以前做原型的時候大部分是用筆和紙，後來換成  Photoshop ，最近逐漸開始轉移到  Sketch 了，這裡要大大的安利一下 Sketch，在移動端的設計上面優勢巨大，資深產品夠確實可以投入 $99。 如果配合  Pixate Studio 還能做些高保真原型（現在已經免費了，感謝 Google）。

團隊溝通方面，之前用 Hangouts （GTalk），後來有機會換成了  Slack ，完全不能自拔（IRC 完全迷）。 產品狗沒有不用腦圖的，不過也不能亂用呀。 MindNode  是我一直不二的選擇，macOS / iOS 全平台同步。

Coding:

  • Terminal / Oh My Zsh / tmux / Git
  • Atom
  • Sequel Pro
  • Dash

因為自己偶而也寫代碼， Oh My Zsh  +  tmux +  Terminal 的 IR_Black 主題就變成了命令行標配（顯得很 geek）。文本編輯器以前是 TextMate （第一個花錢購買的軟件，也是有 7 年歷史了）TextMate 2 支持了中文字符，不用像之前的 hack 中文字體了。最近半年也是轉向了  Atom ，看起來更整體。但是效率其實還是 Textmate 2 更好些，特別是編輯 1W 行以上的文本時，Atom 顯得內存不太夠用。

寫代碼一定離不開  Git ，堅持 Git 一直用命令行提交到 GitHub，各種 Aliases 必須記得牢牢的。大部分時間都用的是 MySQL，所以  Sequel Pro  一定不能錯過，非常簡單高效的軟件，可以讓你完全擺脫 MySQL 官方提供的 GUI。用 Dash  看 API 一定是最方便的，配合 Alfred 基本你都可以不用打開了。偶而在寫 Regex 的時候可以用 Patterns 來幫你，非常好的 Cheatsheet。

Productivity:

  • RescueTime
  • 1Password
  • Dropbox
  • Alfred 2

RescueTime  是一個非常好記錄自己在 Mac、Android 上使用 app、訪問網站時間的記錄器。它真的什麼都記錄，如果有什麼見不得人的訪問，建議不要用。 1Password  是當代人必須使用的密碼管理器，各種網站、app 被拖庫的時候如果在每個地方都使用不同密碼，那你的損失將會非常非常小；雖然 1Password 的替代品蠻多的，用來用去還是它最好用，而且現在很多 app 也支持了。 Dropbox  使用時間就如同它的衰落一樣，越來越邊緣化，雖然我大部分的內容還同步在上面，而且目前並沒有合適的替代品（真可惜）。 Alfred  已經出到 3 了，一直以為 Spotlight 會把它擊垮，但是到現在我還是習慣於 Alfred 和它強大的 Workflow 和第三方的支持， Powerpack 一定要買起，提升生產力刻不容緩。

Reading:

  • Reeder / Pocket
  • Kindle / Duokan Reader

Notes:

  • Evernote
  • Day One

Reeder 是從它在 Beta 的時候就一直在用，以前是配合 Google Reader ，後來把 RSS 同步到 Feedly ，繼續升級 Reeder 從 2.0 到 3.0，macOS / iOS 平台上最好的 RSS 閱讀器（牆裂推薦）。 Pocket 正好配合 Reeder 把看過的最值得收藏的放到 Pocket 裡面，隨時翻看，一般來說我的 Reeder 動不動的就出現了 5000+ 未讀。 Kindle 和 多看閱讀 也是常用的閱讀器，主要是電子書偏向的，中英文很分明，Kindle 看英文，多看看中文。

筆記工具很多，之前用過 Google Keep，Notes，iA Writer，最終還是回歸於  Evernote  和 Day One  了。Evernote 會更偏向於臨時的思路，Day One 是寫更私人的日記、文章的地方，而 Markdown 語法的支持是最重要的。Day One 也不愧是 macOS / iOS 平台上最好的日記 App。Day One 2 增加了多個日記本的功能，非常之實用，本次的分享也在 Day One 2 完成。

GTDs:

  • Reminders
  • Things
  • Tower / Asana / Trello

GTD 也是有太多選擇了，最終還是從繁至簡， Reminders  是最好的提醒選擇，在這個生態系統裡沒有別的可以替代的。個人 Project 我比較喜歡用  Things ，也是從很早的時候接觸的，UI 簡單好用，評價實在高。工作中協作的 GTD 就會選擇  Asana  或者  Tower ，小團隊就喜歡  Trello 。

Music:

  • OctoFM
  • iTunes

工作生活中必不可少的就是音樂了，大部分時間音樂對於我就是陪伴，所以 90% 的時間都用章魚FM（iOS / Android / Mac / Web ）。另外 10% 的時間，我會聽自己的收藏了多年的曲庫，這個曲庫始於 10 多年前的整理，iTunes 是這個世界上最好用最方便建立自己曲庫的軟件，我大約有 200GB 的歌曲，連續聽的話，需要 5 年零兩個月才能完全聽完。

你最理想的工作環境是什麼？ 🔗

  • 可以有陽光，不超過 30 度；也可以下著雨，聽著雨聲；還可以面朝大海，背臨山林。
  • 暢（ke）通（xue）無（shang）阻（wang）的網絡，可以隨時隨地連接世界。
  • 一支筆、一台 MBP，一個筆記本
  • 一碗清茶，一柱香
  • 適合當下的音樂

你平時獲得工作靈感的方式有哪些？ 🔗

  • 洗澡，把自己關進浴室，是我自己自認為最好一個獲得靈感的方式，很多靈感從此而來。
  • 帶著耳機打開後搖／自然聲音的電台，把自己放進音樂的世界。
  • 坐在書房，拿出一支老墨，在端石上打轉。研墨的時候控制自己的呼吸，聆聽墨於石頭的摩擦聲，或者靈感就在其中呢。
  • 研完一池墨，拿出毛筆打開字帖，臨習幾個字，細聽筆墨紙摩擦的聲音，心靜心沉。
  • 冥想打坐，有機會可以獲得靈感。

推薦一件生活中的利器給大家 🔗

智能音箱 方糖 Sugr 。已經用了快一年了，特別喜歡他們的 Slogan「一場手指與音樂的約會」。方糖 Sugr 從設計到用材，從聲音質量到手勢操作都完美的體現了簡潔的概念。讓你所在的每個環境都有適合你的音樂。
